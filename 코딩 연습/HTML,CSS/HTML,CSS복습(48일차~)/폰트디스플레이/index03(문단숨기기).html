한동훈 전 국민의힘 대표 측근인 김종혁 전 최고위원은 28일 CBS라디오 '김현정의 뉴스쇼'에 출연해 지난 26일 홍준표 대구시장이 "한동훈은 윤석열 대통령이 만들어준 인형이다. 들어오면 나한테 죽는다"고 한 발언에 대해 "조폭 두목이 말하는 것 같아 정치인으로서 적절한 얘기는 아닌 것 같다"고 비판했다. 이어 "홍준표 시장 말은 논리적으로 앞뒤가 안 맞는 얘기를 한 것"이라며 "한동훈은 윤석열 인형이라고 해 놓고 한동훈이 윤석열을 쫓아냈다고 하는 건 인형이 주인을 쫓아냈다는 말 아니냐"고 꼬집었다.
