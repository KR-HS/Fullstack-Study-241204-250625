## position
- 키워드
```
static          # 태그가 위에서 아래로 순서대로 배치
relative        # 초기 위치 상태에서 상하좌우로 위치 이동
absolute        # 절대적 위치 좌표 설정 (부모에 realative가 있는 태그 기준으로 움직이게 됨. 없으면 body기준)
fixed           # 화면을 기준으로 절대적 위치 좌표 설정
```

- 포지션이 겹칠때 우선순위 설정
` z-index         # 값이 높을수록 더 앞에 보임. `
